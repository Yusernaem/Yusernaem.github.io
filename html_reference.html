HTML Reference Tags

  1. [!DOCTYPE] = This describes the program being used. Ex - HTML

  2. [html] = Describes the main of the HTML document.

  3. [head] = Defines information about a document.

  4. [body] = Describes the body of the document and what is supposed to be in it. Ex - Header and body text

  5. [a] = This tag describes a hyperlink. Ex - www.youtube.com

  6. [br] = This tag describes a single line break.

  7. [div] = Defines a sention of/in a document.

  8. [font] = describes the desgin of the text being used

  9. [h1 to h6] = These tags define HTML headings.

  10. [hr] = "Horizontal Rule" puts a line across the document or a speciic section.

  11. [img] = Describes an img.

  12. [p] = Describes a paragraph.

  13. [strong] = Defines important text.

  14. [style] = Defines the style information or design informaion for a document.

  15. [table] = Inserts a table that can be customized to get more or less #of columns or rows

  16. [title] = Defines the tile for a document.
